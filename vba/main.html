Visual Basic for Applications is an implementation of Microsoft's event-driven programming language Visual Basic 6,
which was declared legacy in 2008, and is an associated integrated development environment.

Developer: Microsoft
Typing discipline: Static/Dynamic Hybrid, Strong/Weak Hybrid
First appeared: 1993
Stable release: 7.1 (Office 2019)
License: Commercial proprietary software
Paradigm: Multi-paradigm
